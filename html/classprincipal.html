principal::principal ( )

Affiche une vue des valeurs les plus récentes des couples de devises sélectionnés, et un graphique de l'évolution d'un couple de devises sélectionné. A l'initialisation, crée une base de donnée SQLite, envoie une requète HTTP au site du Forex, récupère les infos dans la réponse du Forex et les stocke dans la base de donnée pour les afficher dans l'affichage principale.
