Documentation des fonctions membres

bool CoupleDevise::save ( QSqlDatabase *  db)

Permet de sauver les attributs dans la base de données. Insere les valeurs de tous les attributs dans la base de donnée créée dans la classe 'principal'. La méthode vérifie si les valeurs 'achat' et 'vente' récupérées ont changé depuis le dernier enregistrement de la base de donnée, et crée un nouvel enregistrement uniquement si il y a eu changement.
