Projet FOREX Documentation (paramètres par défaut au bas de la page)

Projet Forex est une application permettant de récupérer en temps réel les cotations de plusieurs couples de devises depuis le web service du Forex et de les afficher dans une interface convivial et pratique. Elle permet aussi de simuler la vente ou l’achat de devises, ainsi que de consulter les cotations d’un couple de devises sur une période déterminée de temps.
Cette documentation s'adresse aux développeurs qui souhaiteraient modifier ou améliorer l’application.



Voici la liste des paramètres avec leur valeur par défaut :
Serveur = "127.0.0.1"
Nom de la base de donnée = "bddForex.db"
Identifiant de la base de données = "admin"
Mot de passe de la base de données = "admin"
Délai entre chaque interrogation du site du Forex = "10000" (valeur en miliseconde)
URL du site du Forex = "http://fxrates.investing.com"
A noter : à l'heure actuelle, le site français ne répond plus.
Voici son url au cas ou il refonctionnerait :
http://fxrates.fr.forexprostools.com

Vous pouvez réinitialisier tous les paramètres à leur valeur par défaut en supprimant le fichier settings.xml à la racine de l'application, ou en décommentant la ligne n°33 du fichier principal.cpp (//settings.clear();)
